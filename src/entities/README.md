# Entities Layer

도메인 모델 및 비즈니스 로직을 담당하는 레이어

## 도메인 목록

### 1. Room (회의방)

회의방의 기본 정보와 상태 관리

### 2. Member (참여자)

User와 Room을 이어주는 관계 도메인

### 3. Message (메시지)

회의 중 주고받는 메시지 (텍스트, 이미지, 음성)

### 4. User (사용자)

로그인한 사용자의 기본 정보

### 5. Note (회의록)

AI가 자동 생성하는 회의록
